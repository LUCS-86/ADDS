# ADDS
GPO FOR ALL THE USERS TO DELETE HISTORY, DELETE DOWNLOAD HISTORY, DELETE CACHE, DELETE COOKIES ETC FROM ANY BROWSER

LOGIN TO DOMAIN ADMIN WHICH HAS ADDS

1) DOWNLOAD .admx and .adml FILE FROM ENTERPRISE DOWNLOAD SECTION OF FOLLOWING BROWER'S WEBSITE
   EXAMPLE FOR MS EDGE:-
a) https://www.microsoft.com/en-us/edge/business/download
b) Looking for an older version of Edge? (YOU WILL GET THIS AFTER SCROLLING DOWN)
c) SELECT THE VERSION BUILD AND PLATFORM
 	FOR VERSION :- C:\Program Files (x86)\Microsoft\Edge\Application (YOU WILL SEE FOLDER WITH VERSION => 143(STABLE 143).0.3650.96 (143.0.3650.96 THIS THE BUILD NUMBER)
d) GET POLICY FILES
e) UNZIP POLICY FILES
f) MicrosoftEdgePolicyTemplates.zip\windows\admx (msedge.admx)
g) MicrosoftEdgePolicyTemplates.zip\windows\admx\en-US (msedge.adml)



Note: FIREFOX WILL DIRECT YOU TO THEIR GIT REPO

2) .admx FILE = IN DESTINATION (C:\Windows\SYSVOL\domain\Policies\PolicyDefinitions)
   .adml FILE = IN DESTINATION (C:\Windows\SYSVOL\domain\Policies\PolicyDefinitions\en-US)

Note: IF FOLDER NOT AVAILABLE MAKE IT

3) CREATE A GPO AND LINK IT IN EITHER DOMAIN OR OU CONTAINING USER IN GPMC (WIN+R => gpmc.msc)

4) RIGHT CLICK GPO_NAME => EDIT

5) USER CONFIGURATION\POLICIES\ADMINISTRATIVE TEMPLATES\SELECT BROWSER (EX: MICROFOFT EDGE)

6) FIND CLEAR BROWISING DATA WHEN MICROSOFT EDGE CLOSES (CLICK ANY FOLDER OR POLICY FILE AND TYPE 'C','L' WITHOUT BREAK YOU WILL FIND IT)

7) ENABLE THE POLICY

8) ENABLE CLEAR CACHED IMAGES AND FILES WHEN MICROSOFT CLOSES (JUST BELOW CLEAR BROWISING DATA ........)

9) WIN+R => GPUPDATE /FORCE

NOTE: IN CHROME YOU HAVE TO TYPE OR ADD THE VALUE. YOU MAY SEE SHOW BUTTON IN OPTION SECTION, CLICK IT AND THE VALUE IS GIVEN IN THE LOWEST PART OF HELP SECTION. VALUES  MUST BE EXACT)

NOTE: IF YOU SET GPO FIRST AND DOWNLOAD BROWSER IN USER LATER THEN RESTARTING  COMPUTER IS COMPULSORY OR GPUPDATE /FORCE IS ENOUGH.
